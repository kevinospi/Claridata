# Flujo de Usuario

## Acceso principal

El usuario accede a la pantalla principal de Claridata.

La interfaz está inspirada en experiencias conversacionales modernas.

---

## Registro

Usuario crea una cuenta o inicia sesión utilizando Google.

---

## Carga de datos

El usuario:

* Arrastra un archivo
* Selecciona un archivo CSV o XLSX

---

## Generación de informe

El sistema:

* Analiza el dataset
* Genera estadísticas
* Genera visualizaciones
* Genera interpretaciones
* Genera hallazgos

---

## Exploración del informe

El usuario puede:

* Leer el resumen ejecutivo
* Revisar gráficas
* Revisar interpretaciones
* Revisar conclusiones

---

## Conversación

El usuario puede realizar preguntas relacionadas con el informe.

La conversación mantiene contexto únicamente para el informe actual.

---

## Guardado

El usuario decide si desea guardar el informe.

El sistema no guarda informes automáticamente.

---

## Exportación

El usuario puede exportar el informe en formato PDF.
